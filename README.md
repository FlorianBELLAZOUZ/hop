Hop -- Multitier JavaScript
===========================

### To build:

Prerequisites:

 * GCC
 * Bigloo 4.2a or newer <http://www-sop.inria.fr/indes/fp/Bigloo>
 * GNU Make 3.81 or newer
 * Autoconf/Automake/Libtool
 * OpenSSL 1.0.0 or newer

Unix/Machintosh:

```sh
./configure
make
make install
```

### To run the tests:

```sh
make test
```

### To build the documentation:

```sh
make doc
```
