A basic Hop.js example showing how to implement a server-side
file browser.

To install:

```
npm install
```

To run:

```
npm run server
```

To run the typed version:

```
npm run serverts
```

To browse:

```
http://localhost:8888/filebrowser
```

To browse the typed version:

```
https://localhost:4000/filebrowser
```
